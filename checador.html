Manual de Usuario

Sistema de Control de Entradas y Salidas

Checador (Pantalla con NUA)

Esta es la pantalla principal que utilizarán los empleados para registrar el inicio y el fin de su jornada laboral.

Funcionalidad

  • Campo "INSERTE SU NUA": Aquí el empleado debe escribir su NUA (Número Único de Alumno/Empleado), que es el identificador personal e intransferible para cada usuario en el sistema.
  • Botón "ACEPTAR": Una vez ingresado el NUA, el empleado debe presionar este botón. El sistema registrará automáticamente la fecha y hora, guardando el registro como una entrada o una salida.

Inicio de Sesión

Esta pantalla permite el acceso seguro a las funciones administrativas del sistema y está dirigida únicamente a personal autorizado.

Funcionalidad

  • Campo "CORREO": Ingrese la dirección de correo electrónico que tiene registrada en el sistema.
  • Campo "CONTRASEÑA": Ingrese su contraseña personal. Por seguridad, los caracteres no se mostrarán.
  • Botón "INICIAR SESIÓN": Al hacer clic, el sistema validará sus credenciales. Si son correctas, se le dará acceso al panel de administración.

Registro de Empleado

Este módulo se utiliza para dar de alta a nuevos colaboradores en la base de datos del sistema.

Funcionalidad

  • Formulario de registro: Rellene cada uno de los campos con la información del nuevo empleado.
  • Botón "REGISTRAR": Una vez que todos los campos estén completos, presione este botón para guardar el nuevo registro en el sistema.

Descripción de Cargo

Esta sección es fundamental para definir y administrar los diferentes puestos de trabajo que existen.

Funcionalidad

  • Campos de información: Complete los datos para definir un cargo, como el Nombre del cargo, departamento y requisitos.
  • Botón "Guardar": Presione para crear una nueva descripción de cargo o salvar los cambios.
  • Botón "Editar": Permite modificar la información de un cargo previamente seleccionado.
  • Botón "Eliminar": Elimina de forma permanente la descripción de un cargo. Use con precaución.

Lista de Empleados

Ofrece una vista general de todo el personal registrado en el sistema.

Funcionalidad

  • Tabla de empleados: Muestra información clave de cada empleado (NUE, nombre, cargo, etc.).
  • Botón "REPORTE": Genera un documento con la lista completa de los empleados.
  • Botón "ACTUALIZAR": Refresca la lista para mostrar los cambios más recientes.
  • Botón "ELIMINAR": Permite dar de baja a un empleado seleccionado en la lista.

Lista de Incidencias

Permite registrar, consultar y administrar los eventos que afecten la asistencia de un empleado.

Funcionalidad

  • Lista de Incidencias: Muestra un historial de todas las incidencias registradas.
  • Reporte de Incidencias: Permite registrar una nueva incidencia.
  • Campos de Fecha e Incidencia: Seleccione la fecha y el tipo de incidencia (Llegada tarde, Permiso, etc.).
  • Campo de Observaciones: Agregue cualquier detalle o justificación relevante.
  • Botón "Guardar": Registra la nueva incidencia en el historial del empleado.
  • Botón "Reporte": Genera un comprobante o reporte formal de la incidencia creada.

Lista de Asistencias

Herramienta principal para consultar los registros de entrada/salida y generar reportes.

Funcionalidad

  • Tabla de asistencias: Muestra de forma cronológica los registros de checador de cada empleado.
  • Botón "REPORTE": Permite generar un reporte de asistencias filtrado (por semana, quincena, etc.) para fines de nómina o seguimiento.
